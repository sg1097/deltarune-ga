# Irish Translation for Deltarune
A work in progress Irish translation for Deltarune Ch1. I just made this for fun and to practice my Irish a bit so the translations are quite poor.  

# Note for ISE
lang folder contains all strings that I translated and sprites that I customised. You can ignore the fonts folder  
gml files contains the code files that I edited  
You can ignore Ch1 data it's only there to make installing easier  

# How to Install
Have a copy of Deltarune. Having chapters 3 and 4 is preferable so some lesser used fonts support accents/fadas but the demo version should still work just fine  
Download the Ch1 data and lang folders from this page  
Download and install Deltranslate from https://github.com/Lazy-Desman/DeltranslatePatch/releases  
This extracts all the game data that needs to be translated from the data.win files  
Copy the contents of my lang folder to the one in your game folder and replace all  
The game should be fully playable from this point but if you want some extra polish, copy the data.win from the Ch1 data folder into your chapter1_windows folder and replace the original  
If you are familiar with Undertale Mod Tool or similar programs you could try replace the individual gml files I changed instead but since Chapter 1 is free anyway I figured the data.win replacement was quicker
